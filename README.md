# mika-star-fishing-py

一个用于《星痕共鸣》游戏的自动化钓鱼辅助工具，使用 Python 和 PyAutoGUI 实现自动化操作。

<img width="1803" height="1412" alt="G8BB4EJ 1YBU2`0SYI~Z_8J" src="https://github.com/user-attachments/assets/1dbdfd5e-ea27-4b80-b286-27fb97f65f5e" />



## 功能特性

- 🎣 **自动钓鱼**：支持全自动钓鱼流程
- 🛒 **自动购买**：鱼饵和鱼竿用完后自动前往商店购买
- 🔄 **自动续钓**：钓鱼成功后自动继续下一轮
- 📅 **智能处理**：自动关闭第二天月卡签到确认弹框
- ⚙️ **灵活配置**：支持选择不同品质的鱼饵和鱼竿

## 使用要求

### 系统要求
- Windows 操作系统
- Python 3.x 环境（如需运行源码）
- 屏幕分辨率：**3840x2160（4K）**（暂未在其他分辨率下测试）

### 依赖库
运行源码需要安装以下 Python 库：
```bash
pip install pyautogui pywin32 keyboard numpy
```

## 使用方法

### 方式一：直接运行 EXE（推荐）

1. 下载 `dist/mika-fishing.exe` 文件
2. 双击运行程序
3. 在游戏窗口中选择合适的鱼饵和鱼竿类型
4. 点击"开始钓鱼"按钮
5. 按 `ESC` 键或点击按钮可随时停止

### 方式二：运行 Python 源码

1. 确保已安装所有依赖库
2. 运行主程序：
   ```bash
   python fishing.py
   ```
3. 按照 GUI 界面提示操作

## 使用说明

### 配置选项

- **优先购买鱼饵**：可选择普通鱼饵、不空军鱼饵、特制鱼饵
- **优先购买鱼竿**：可选择普通鱼竿、强韧鱼竿、柔韧鱼竿

### 操作提示

- 程序启动后会自动移动鼠标到屏幕中心并点击开始
- 在钓鱼过程中会实时显示状态信息
- 自动检测鱼饵/鱼竿不足并前往商店购买
- 自动识别左右拉鱼提示并进行相应操作

### 注意事项

⚠️ **重要提示**：
- 程序仅在 **3840x2160** 分辨率下测试通过
- 左右溜鱼操作有一定延迟，稳定性不是百分百，但一般紫鱼都可以钓到😁
- 建议在使用前先测试程序功能是否正常
- 使用过程中请勿移动鼠标或操作键盘，以免干扰自动化流程
- 请尽量买足鱼饵和鱼竿，保证长时间挂机不断档

## 已知问题

- 左右拉鱼操作存在一定延迟，可能影响部分高端鱼类的成功率
- 仅在 4K 分辨率下测试，其他分辨率可能出现识别问题

## 技术说明

- **主要技术**：Python + PyAutoGUI
- **图像识别**：使用 PyAutoGUI 的图片识别功能进行游戏界面元素定位
- **自动化控制**：模拟鼠标点击和键盘按键操作
